1942 Aircraft War Game

Introduction

This is my first game made by myself. So it is naive and really simple to implement. I just use PyGame. The rule of the game is quite easy: when someone play a card, another one next to him must play a card with the same suit or number, or you need to keep one card. The one who has the smallest sum of keeped cards' number wins. Attention, jokers and "2"s can change the suit when you play it. Before play the game, you need to install python and pygame. Run main.py to start the game.
